# 📱 Mobile Sales Analytics Dashboard

> A comprehensive **Power BI Dashboard** designed to analyze mobile phone sales performance across different brands, cities, payment methods, and time periods. The dashboard provides interactive insights into sales trends, customer purchasing behavior, and year-over-year business performance.

---

# 📊 Project Overview

The **Mobile Sales Analytics Dashboard** helps businesses monitor overall sales performance, product demand, transaction trends, and regional sales distribution through interactive Power BI visualizations.

The report consists of **three interactive dashboard pages**:

- 🏠 Dashboard Overview
- 📈 Month-to-Date (MTD) Report
- 📊 Same Period Last Year (SPLY) Analysis

These dashboards enable stakeholders to track KPIs, compare historical performance, and make data-driven business decisions.

---

# ✨ Features

## 📌 Dashboard Overview

Provides a complete snapshot of business performance.

### Key KPIs

- 💰 Total Sales
- 📦 Total Quantity Sold
- 💳 Total Transactions
- 💲 Average Selling Price

### Sales Insights

- 🗺️ Total Sales by City (Map)
- 📈 Monthly Sales Quantity Trend
- 📱 Sales by Mobile Model
- 🏷️ Sales by Brand
- 📅 Sales by Day of Week

### Customer Insights

- ⭐ Rating Status Distribution
- 💳 Transactions by Payment Method

### Interactive Filters

- Mobile Model
- Brand
- Payment Method
- Month Selection

---

## 📈 Month-to-Date (MTD) Report

Track cumulative sales performance throughout the selected month.

### Includes

- 📈 Month-to-Date Sales Trend
- 📊 Running Sales Performance
- 📅 Daily Sales Growth
- 📌 Monthly KPI Summary
- 📆 Dynamic Month Selection
- 📂 Year Filter

---

## 📊 Same Period Last Year (SPLY) Analysis

Compare current sales performance with the same period of the previous year.

### Includes

- 📊 Year-over-Year Sales Comparison
- 📅 Quarterly Sales Comparison
- 📈 Monthly Sales Comparison
- 📋 Historical Sales Table
- 📉 Previous Year Sales Analysis

---

# 📊 Key Performance Indicators (KPIs)

| KPI | Description |
|------|-------------|
| Total Sales | Overall revenue generated from mobile sales |
| Total Quantity | Number of mobile devices sold |
| Transactions | Total completed sales transactions |
| Average Price | Average selling price per mobile device |
| Sales by City | Regional sales performance |
| Sales by Brand | Brand-wise sales analysis |
| Sales by Mobile Model | Best-selling mobile models |
| Payment Method | Customer payment preferences |
| Rating Status | Customer satisfaction analysis |
| MTD Sales | Month-to-Date cumulative sales |
| Same Period Last Year | Year-over-Year sales comparison |
